# ASCII Котик Generator

Приложение на Python3 для генерации ASCII-графики котика с помощью LLM моделей по OpenAI протоколу.

## Описание

Приложение отправляет запрос к LLM модели с системным промптом на русском языке, который заставляет модель создать красивый и подробный ASCII рисунок котика, говорящего "Привет, Мир!".

## Функционал

- Загрузка настроек из конфигурационного файла `settings.toml`
- Формирование русских промптов для генерации ASCII-графики
- Отправка запросов к LLM моделям по OpenAI протоколу
- Вывод результата на экран

## Требования

- Python 3.7+
- Библиотеки: `requests`, `toml`

## Установка

1. Склонируйте репозиторий:
```bash
git clone <repository-url>
cd day-01
```

2. Установите зависимости:
```bash
pip install requests toml
```

3. Настройте конфигурационный файл `settings.toml`:
```toml
base_url = "https://api.example.com/v1"
api_key = "ваш-api-ключ-здесь"
model = "gpt-3.5-turbo"
timeout = 30
max_tokens = 1000
```

## Использование

Запустите приложение:
```bash
python main.py
```

Приложение автоматически:
- Загрузит настройки из `settings.toml`
- Сформирует промпты на русском языке
- Отправит запрос к LLM модели
- Выведет ASCII-графику котика на экран

## Конфигурация

Файл `settings.toml` содержит следующие параметры:

- `base_url` - URL endpoint API
- `api_key` - API ключ для доступа к модели
- `model` - название модели (например, gpt-3.5-turbo)
- `timeout` - время ожидания ответа в секундах
- `max_tokens` - максимальное количество токенов в ответе

## Промпты

Приложение использует два промпта на русском языке:

1. **Системный промпт**: Устанавливает роль модели как художника ASCII-графики
2. **Пользовательский промпт**: Запрос на создание ASCII котика с фразой "Привет, Мир!"
