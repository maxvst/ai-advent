# Temperature Demo

Node.js + TypeScript приложение для демонстрации влияния параметра `temperature` на ответы LLM.

## Возможности

- Отправка одного и того же запроса 3 раза с разными температурами: 0, 0.7, 1.2
- Вывод всех ответов в консоль с визуальным разделением
- Сохранение результатов в файл `results.json`
- Настройки API вынесены в отдельный конфигурационный файл

## Установка

```bash
npm install
```

## Настройка

Отредактируйте файл `src/config/api.ts`:

```typescript
export const apiConfig = {
  apiKey: 'YOUR_API_KEY',           // Ваш API ключ
  baseURL: 'https://openrouter.ai/api/v1',  // URL API
  model: 'deepseek/deepseek-v3.2',  // Модель
  maxTokens: 1000,
};
```

Или используйте переменную окружения:

```bash
export OPENAI_API_KEY=your_api_key
```

## Запуск

```bash
npm start
```

## Результаты

После запуска создаётся файл `results.json` со структурой:

```json
{
  "prompt": "Напиши 5 названий для кофейни",
  "model": "deepseek/deepseek-v3.2",
  "results": [
    {
      "temperature": 0,
      "response": "...",
      "timestamp": "..."
    },
    {
      "temperature": 0.7,
      "response": "...",
      "timestamp": "..."
    },
    {
      "temperature": 1.2,
      "response": "...",
      "timestamp": "..."
    }
  ]
}
```

## Температура

- **0** — Детерминированный, предсказуемый ответ
- **0.7** — Баланс между креативностью и предсказуемостью
- **1.2** — Максимальная креативность и непредсказуемость

## Анализ результатов

### Температура = 0

| Критерий | Значение |
|----------|----------|
| Названия | «Уголок», «Чашка & Мысли», «Зерно», «На бодрость», «Aroma Terra» |
| Стиль | Минималистичный, лаконичный |
| Форматирование | Простое, без markdown-разметки |
| Описания | Краткие и прямолинейные |

### Температура = 0.7

| Критерий | Значение |
|----------|----------|
| Названия | «Зерно & Пена», «Уголок для разговоров», «Daily Brew», «Созрело для кофе», «Ароматия» |
| Стиль | Разнообразный, появились английские названия |
| Форматирование | Используется markdown (###) |
| Описания | Более развёрнутые, с обоснованием концепции |

### Температура = 1.2

| Критерий | Значение |
|----------|----------|
| Названия | «Чашка и Тень», «Кофеограф», «Молотый День», «Зерновая Лаборатория», «Балкон» |
| Стиль | Наиболее креативный, сложные составные названия |
| Форматирование | Эмоциональное, с восклицаниями |
| Описания | Самые длинные и поэтичные |

### Выводы

1. **С ростом температуры** увеличивается разнообразие и креативность названий
2. **При температуре 0** названия более консервативные и понятные («Зерно», «Уголок»)
3. **При температуре 1.2** появляются неологизмы («Кофеограф») и сложные метафоры
4. **Форматирование** становится богаче с увеличением температуры
5. **Объём ответов** также растёт с температурой — при 1.2 описания наиболее развёрнутые
